# MiningSummarySystem
## 概要
マインクラフト統合版においてプレイヤーのブロック破壊を検知・集計するアドオンです。
一部ブロックを除いたすべてのブロックの破壊アクションを集計します。（クリエイティブモードを除く）

## 機能
### ・ワールド統計
すべてのプレイヤー合算のブロック破壊数及びブロックごとの破壊数を集計します。\
/function summary を実行することで統計情報を表示します（OP限定）

### ・ランキング
集計情報からプレイヤーごとの総破壊数ランキングを作成します。
コンパス（minecraft:compass）を右クリック（使用）することで破壊数TOP10のプレイヤーと使用者の順位及び破壊数を表示します。\
/function rank を実行することで同じ操作を行えます（OP限定）

### ・アクションバー表示
プレイヤーごとの破壊数, 順位, 次の順位までの必要破壊数をアクションバーに表示します。\
時計（minecraft:clock）を右クリック（使用）することで表示のON/OFFを切り替えます。

### ・スコアリセット
/function reset を一定時間内に２回実行することですべての統計情報をリセットします。（OP限定）\
※リセットの巻き戻しはできません※

### ・ログ表示
タグ"logListener"が付与されているプレイヤーに対しログを表示します。\
対象イベント：エラー, 初めてのワールド参加, ブロック破壊, コンパス・時計使用, コマンド使用
